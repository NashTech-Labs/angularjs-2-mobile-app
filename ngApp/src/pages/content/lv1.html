 We will use typescript further in examples, so

Typescript:
Typescript is Microsoft extension of javascript that allows to use of all ES2015 features. Like type checking and object oriented features. Extension for typescript file is .ts, know more on typescriptlang.org.

Transpiling :
Our browser doesn't know to read typescript so we have to convert it into javascript, we have two options:
1- transpile to javascript in the browser.
2- transpile to javascript before shipping to browser.
We're choosing with the second option becuase we don't want to overhead the browser work and delay loading our app on user end.


Hello AngularJs-2 we have to print this in browser using AngularJs-2 so lets create our first Ng2 app.
We'll create 3 basic examples

1- In this example we will render our html inside the template under Component.
Let's start with the Component.

app.component.ts
.import {Component} from '@angular/core';
@Component({;
selector: 'first-app',
template: '<h1>Hello AngularJs-2</h1>'
})
Component is the basic block of Ng2 app as like controller in Ng1.x.
The basic structure of application you have already seen in 5min Quickstart.
main.ts
This file is the starting point of our application, because it starts/boost our app.ts file, which is first file in our case.
import {bootstrap} from '@angular/platform-browser-dynamic';
import {AppComponent} from './app.component';
bootstrap(AppComponent);
2- Now we will saperate the inline html into another html file and will call it in app.component.ts, rest of the things will be same
import { Component } from '@angular/core';
@Component({
selector: 'my-app',
templateUrl: 'index.html'
})
export class AppComponent{ }

3- Now put some dynamic data in the template so for that add an attribute in class AppComponent, like:
@Component({
selector: 'my-app',
template: '<h2> { { title } } </h2>'
})
export class AppComponent{
title = ”Hey this is dynamic Hello NG2”.
}
