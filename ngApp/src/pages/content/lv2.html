Directives, Pipes & Methods


Directive: A directive (in angular) is how we add dynamic behavior to html.

Basically kinds of three:
1- Component: (has a template so kind a directive)
2- Structural Directive
3- Attribute Directives

2)- Structural Directive:

Alters layout by adding, removing or replacing HTML elements. ie. ngIf, ngFor etc.
Example for Structural Directives: ngIf and ngFor

import { Component } from '@angular/core';

@Component({
selector: 'my-app',
template: `
<h1> Hey Welcome to : inside double curly braces put : title </h1>
})
export class AppComponent {
title = "Bike Rash";

carParts = [{
"id":1,
"name": "SuperTyres",
"description":"Very best rubber typres",
"inStock": 4,
“price”: 94.7988
},
{
"id":2,
"name": "Super Seat",
"description":"Very best seat",
"inStock": 5,
“price”: 28.9808
},
{
"id":3,
"name": "Super Light",
"description":"This is really powered light",
"inStock": 0,
“price”: 24.9808
}];
}
Here if you notice we have used ngIf and ngFor. The usage are simple to understand.

Using ngIf what we are doing here is just replacing the, “0” from stock availablility by “Out of stock”. ngFor is used to repeat the object array.

Methods:
For using methods lets create a scenario to count the total number of bikeParts. The json data we created above contains 3 bike parts as we know.

import { Component } from '@angular/core';
@Component({
selector: 'my-app',
template: `<h1>Hey Welcome to : inside double curly braces put : title </h1>
export class AppComponent {
title = "Bike Rash";
bikeParts = [{...}];
totalBikeParts(){
let sum = 0;
for(let totalParts of this.bikeParts){
sum = sum + totalParts.inStock;
}
return sum;
}
}

Total 9 Bike Parts are available
